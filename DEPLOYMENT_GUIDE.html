🚀 Deployment Options

Option 1: Vercel (Recommended)

  1. Go to vercel.com
  2. Sign up/Login (free account)
  3. Click "Add New Project"
  4. Upload the epa-mining-simple.zip file
  5. Your app will be live instantly!

Option 2: Render

  1. Go to render.com
  2. Create free account
  3. Choose "Static Site"
  4. Upload your files

Option 3: Firebase Hosting

  1. Go to firebase.google.com
  2. Create project
  3. Enable Hosting
  4. Upload the dist folder contents
